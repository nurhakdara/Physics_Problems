# Exercise 3: Superposition Principle (Easy Explanation)

## Given Waves

y₁(x,t) = A sin(kx − ωt)
y₂(x,t) = A sin(kx + ωt)

---

#  What do these variables mean?

* y(x,t): displacement of the wave (how much it moves up/down)
* A: amplitude (maximum height of the wave)
* k: wave number → relates to wavelength
  k = 2π / λ
* ω: angular frequency → relates to frequency
  ω = 2πf
* x: position along the string
* t: time

---

#  Step 1: What is happening physically?

* First wave → moving to the right
* Second wave → moving to the left
* Same amplitude and frequency

 When they meet → they form a **standing wave**

---

#  Step 2: Add the waves

y = y₁ + y₂

y = A sin(kx − ωt) + A sin(kx + ωt)

---

#  Step 3: Use identity

sin(a − b) + sin(a + b) = 2 sin(a) cos(b)

---

#  Resulting Wave

y(x,t) = 2A sin(kx) cos(ωt)

---

# 🔍 What does this result mean?

The equation has **two parts**:

### 1️ sin(kx) → depends on position

* tells us WHERE the wave moves
* determines nodes and antinodes

### 2️ cos(ωt) → depends on time

* tells us HOW it moves over time

---

# Step 4: Find Nodes

Nodes = points that NEVER move

So:

y(x,t) = 0 for all time

This happens when:

sin(kx) = 0

---

## Solve:

kx = nπ

x = nπ / k

---

## Replace k:

k = 2π / λ

So:

x = nλ / 2

---

# Final Answers

### Standing Wave:

y(x,t) = 2A sin(kx) cos(ωt)

---

### Node Positions:

x = nλ / 2

(where n = 0, 1, 2, 3, ...)

---

#  Intuition

* Waves cancel each other at nodes → no motion
* Waves add up at antinodes → maximum motion
* The pattern does NOT move → it "stands"
