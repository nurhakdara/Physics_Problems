# Exercise 4: Phase Difference

## Problem

Find the phase difference (in radians) between two points separated by:

Δx = λ / 3

---

#  Step 1: Key Formula

Phase difference is:

Δφ = (2π / λ) × Δx

Where:

* Δφ = phase difference (radians)
* λ = wavelength
* Δx = distance between points

---

# 🔹 Step 2: Substitute Values

Δx = λ / 3

So:

Δφ = (2π / λ) × (λ / 3)

---

# 🔹 Step 3: Simplify

λ cancels:

Δφ = 2π / 3

---

#  Final Answer

Δφ = 2π / 3 radians

---

# Interpretation

* One full wavelength → 2π radians
* λ/3 → one-third of a cycle

So:

phase = (1/3) × 2π
